Портфолио

LitNet

LitNet — это простая и удобная в использовании платформа для обмена книгами. Здесь можно найти любой из множества жанров отечественной и зарубежной литературы. Есть функционал загрузки и скачивания книг, просмотр PDF-файлов, авторизация и регистрация, возможность смены данных в личном кабинете, поиск книг по автору и названию, фильтр по жанрам.

Чат-бот для розыгрыша по телефону

CallPrankBot — это чат-бот, созданный на основе Telegram API, с помощью которого можно разыграть своих друзей и знакомых необычным звонком-розыгрышем.

Сайт-визитка с адаптивной вёрсткой

Сайт-визитка для образовательного проекта. Адаптирован под мобильные устройства, состоит из пяти простых блоков.

Парсер заданий ЕГЭ, запись в БД MySQL

После предварительной авторизации скрипт циклически проходит по вариантам с сайта "Решу ЕГЭ" по русскому языку, скрапит текст заданий и ответ, сохраняет данные в Базу Данных MySQL.
